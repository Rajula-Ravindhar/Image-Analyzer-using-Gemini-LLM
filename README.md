# Image-Analyzer-using-Gemini-LLM

This project is a simple yet powerful Streamlit-based web application that performs image analysis using **Google's Gemini Vision LLM**. 
Users can upload an image and provide a text prompt, and the model generates an intelligent response based on the visual contents of the image and the given prompt.
